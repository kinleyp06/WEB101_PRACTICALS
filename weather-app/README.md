# Weather API Dashboard

A simple app to check weather and save locations using GET, POST, PUT, DELETE requests.

## Files
- `index.html` - The webpage
- `script.js` - The code

## Setup

1. Get a free API key from https://openweathermap.org/
2. Open `script.js` and replace this line:
```javascript
const WEATHER_API_KEY = 'API Key'; // Replace with your API Key
```
3. Open `index.html` in your browser

## How to Use

**GET Weather** - Type a city name, click Get Weather  
**POST Location** - Fill out the form, click Save Location  
**PUT Update** - In Saved Locations, click Edit on any item  
**DELETE Remove** - In Saved Locations, click Delete on any item  

## Notes
- New API keys can take an hour to activate
- The fake API resets on page refresh
- Country codes work best as two letters (US, GB, JP)
